## OTA v1.0.7 以前 {#ota-v1-0-7-以前}

**ステータス: 修正済み（要 再インストール）**

2026-05-15 以前のファームウェア（v1.0.7 以前）には OTA 機能をブロックする重大なバグがありました。該当バージョンの本体は今後の OTA アップデートを受け取れません。

- 該当 device の見分け方: バージョン表示が `1.0.7` 以下 / OTA ボタンを押しても進まない / 「Preparing 8%」で止まる。
- 対処: PC の WEB インストーラから配信中の最新版を USB 経由で新規インストールしてください。詳しくは [更新情報の告知](../updates/#配信ステータス-2026-06-21-時点) を確認してください。
- v1.0.10 以降の本体は OTA で最新版に更新可能です（再インストール不要）。

## GPS NoDATA {#gps-nodata}

**ステータス: 修正済み（v1.0.15）**

v1.0.14 では、電源再投入後に GPS 受信機の設定が失われ、GPS が NoDATA と表示される場合がありました。v1.0.15 以降への OTA 更新で自動回復します。

更新後も改善しない場合の切り分け手順は [トラブル対応「③GPS が NoDATA と表示される」](../troubleshooting/#_3gpsがnodataと表示される) にまとめています。

## Tab5 Wi-Fi AP {#tab5-wi-fi-ap}

**ステータス: 回避策あり（v1.0.18 beta で改善）**

M5Stack Tab5 で "Kuruma-Logger" AP は見えるが、iPhone/スマホ側が「接続中」のまま完了しないことがありました。v1.0.18 で電源再投入・USB 抜き差し後の AP 起動、STA 接続失敗時の AP 維持、iPhone の接続確認応答を改善しています。

改善後も発生する場合の確認手順は [トラブル対応「⑤Tab5 でスマホが Wi-Fi AP に接続完了しない」](../troubleshooting/#_5tab5でスマホがwi-fi-apに接続完了しない) を確認してください。

## Tab5 旧配線 {#tab5-旧配線}

**ステータス: 回避策あり（ハーネス確認が必要）**

v1.0.14 で Tab5 の GNSS/CAN ピン配置を変更しました。v1.0.13 以前から配線済みの場合、ファームウェアの更新だけでは解決せず、ハーネス側の配線確認が必要です（GNSS は M-Bus pin 21/22）。

確認手順は [トラブル対応「③GPS が NoDATA と表示される」の手順3](../troubleshooting/#_3gpsがnodataと表示される) を参照してください。

## アプリ内ブラウザログイン {#アプリ内ブラウザログイン}

**ステータス: 回避策あり**

X などのアプリ内ブラウザからマイページや WEB インストーラを直接開くと、Google ログインの画面遷移や認証情報の保存が制限され、エラーになる場合があります。

回避策は [トラブル対応「④マイページにログインできない」](../troubleshooting/#_4マイページにログインできない) にまとめています（Safari / Chrome / Edge などの通常ブラウザで開き直してください）。

## 走行中の電源断 {#走行中の電源断}

**ステータス: 回避策あり**

ミニ CAN ユニットと中間ハーネスの接続部（ネジ固定部）が緩んでいると、走行中に M5Stack の電源が落ちることがあります。

対処方法は [トラブル対応「①走行中に M5Stack の電源が落ちる」](../troubleshooting/#_1走行中にm5stackの電源が落ちる) を参照してください。

## ミニCANユニット固着 {#ミニcanユニット固着}

**ステータス: 回避策あり（配線加工が必要）**

電源断が起きた際、ミニ CAN ユニットが送信状態で固着し、車両の CAN バスに意図しない操作が行われることでメータに警告灯が表示される場合があります。ミニ CAN ユニットと M5Stack 間の Grove ケーブルの TX ラインを切除することで防止できます（切除方法は [セットアップページ](../setup/#ハードウェアの組み立て) に記載）。

詳細は [トラブル対応「②走行中に M5Stack の電源が落ちて、メータに警告灯が表示される」](../troubleshooting/#_2走行中にm5stackの電源が落ちて、メータに警告灯が表示される。) を参照してください。
