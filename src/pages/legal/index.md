# 特定商取引法に基づく表記

Kuruma-Logger のライセンス販売に関する表示事項です。

以下のうち、個人情報保護の観点で公開を省略している項目は、請求があった場合に遅滞なく開示します。

| 項目 | 内容 |
| --- | --- |
| 販売事業者名 | 高倉涼 |
| 運営責任者 | 高倉涼 |
| 所在地 | 請求があった場合、遅滞なく開示します。 |
| 電話番号 | 請求があった場合、遅滞なく開示します。 |
| メールアドレス | kurumario.nd5rc@gmail.com |
| 販売価格 | 各商品ページに税込価格を表示します。 |
| 商品代金以外の必要料金 | インターネット接続料金、振込手数料等は購入者負担です。 |
| 支払方法 | クレジットカード決済（Stripe） |
| 支払時期 | 注文確定時に決済されます。 |
| 商品の引渡時期 | 決済完了後、画面表示またはメールにてライセンス情報を案内します。 |
| 商品の提供形態 | デジタルコンテンツ（ライセンスキーの提供） |
| 返品・キャンセル | お客様都合による購入後の返品・返金は原則不可です。販売者起因の不具合・ライセンス発行不備が確認できた場合は、再発行または返金で個別対応します。 |
| 動作環境 | 対応環境はセットアップページに記載します。 |

## 開示請求の受付方法

住所・電話番号の開示請求は、上記メールアドレス宛に受け付けます。

- 件名に「開示請求」と記載してください。
- 本文に、氏名・連絡先・請求項目を記載してください。
- 請求内容を確認後、遅滞なく返信します。
