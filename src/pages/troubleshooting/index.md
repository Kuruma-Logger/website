# トラブル対応（FAQ統合）

症状別の確認手順と、よくある質問を1ページに統合しています。

## 症状別の確認手順

### CAN 値が取れない

1. 配線極性と接触を確認
2. 車種プロファイル設定を確認
3. 電源再投入後に再確認

### GPS が安定しない

1. アンテナ配置を見直す
2. 受信状況が開けた環境で確認
3. 初期化 UI から再設定

### Wi-Fi 接続できない

1. SSID/パスワードを再確認
2. 2.4GHz 帯の利用可否を確認
3. 端末側ネットワーク設定を再作成

### SD 保存が失敗する

1. カード差し込みと規格を確認
2. 空き容量と破損有無を確認
3. 別カードで再現性を確認

## よくある質問（FAQ）

### 起動したのにデータ表示が更新されません

CAN 配線、車種設定、電源供給状態を順に確認してください。まずは配線接触不良の有無を確認するのが早いです。

### GPS が捕捉されるまでどのくらいかかりますか？

環境により数十秒から数分かかる場合があります。空が開けた場所でしばらく待ってください。

### RaceChrono 連携の確認ポイントは？

Wi-Fi 接続先、TCP 配信設定、アプリ側の接続先設定を一致させてください。

### MQTT 配信が安定しない場合の対処は？

アクセスポイントの電波状況、Broker 側設定、再接続ログを確認し、切断周期があるかを確認してください。

### SD 保存が開始されません

SD マウント状態、空き容量、ライセンス機能設定を確認してください。

### ログの一部が欠けて見えることがあります

記録中の電源瞬断やカード速度不足が原因になることがあります。別カードで再現確認してください。

## 問い合わせ前に準備する情報

- 車種（ND / FL5）
- ファームウェアバージョン
- 発生手順（再現する操作）
- ログやスクリーンショットの有無

準備できたら [問い合わせフォーム](../contact/) から送信してください。
